p5js
====

`โมดูล p5 <p5/index.html>`_

ฟังก์ชั่นหลักของ p5js ประกอบด้วย ฟังก์ชั่นเกี่ยวกับ การแสดงผลแสงสีบนจอภาพ การควบคุมอุปกรณ์อย่างเมาส์และคีย์บอร์ด รวมไปถึงฟังก์ชั่นช่วยเหลือด้านการคำนวณทางคณิตศาสตร์


`โมดูล p5.dom <p5.dom/index.html>`_

ฟังก์ชั่นเสริมที่ช่วยให้เราสื่อสารกับ HTML5 object ที่อยู่นอกเหนือจาก canvas เช่น ปุ่มกด, วิดีโอ, เว็บแคม
