bezierDetail()
==============

Sets the resolution at which Beziers display.
The default value is 20.

**รูปแบบการใช้งาน**

bezierDetail ( detail )

**พารามิเตอร์**

- ``detail``  Number: resolution of the curves


.. raw:: html

	<script type="text/p5" data-autoplay data-hide-sourcecode>
	background(204);
	bezierDetail(50);
	bezier(85, 20, 10, 10, 90, 90, 15, 80);

	</script>
