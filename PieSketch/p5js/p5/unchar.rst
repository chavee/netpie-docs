unchar()
========

Converts a single-character string to its corresponding integer
representation. When an array of single-character string values is passed
in, then an array of integers of the same length is returned.

**รูปแบบการใช้งาน**

unchar ( n )

**พารามิเตอร์**

- ``n``  : value to parse
