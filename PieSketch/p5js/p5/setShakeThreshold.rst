setShakeThreshold()
===================

The setShakeThreshold() function is used to set the movement threshold for
the deviceShaken() function. The default threshold is set to 30.

**รูปแบบการใช้งาน**

setShakeThreshold ( value )

**พารามิเตอร์**

- ``value``  number: The threshold value
