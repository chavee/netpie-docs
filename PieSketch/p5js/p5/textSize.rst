textSize()
==========

Sets/gets the current font size. This size will be used in all subsequent
calls to the text() function. Font size is measured in pixels.

**รูปแบบการใช้งาน**

textSize ( theSize )

**พารามิเตอร์**

- ``theSize``  : the size of the letters in units of pixels
