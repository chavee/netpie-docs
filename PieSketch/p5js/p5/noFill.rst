noFill()
========

Disables filling geometry. If both noStroke() and noFill() are called,
nothing will be drawn to the screen.

**รูปแบบการใช้งาน**

noFill ( )

.. raw:: html

	<script type="text/p5" data-autoplay data-hide-sourcecode>
	rect(15, 10, 55, 55);
	noFill();
	rect(20, 20, 60, 60);

	</script>
