textLeading()
=============

Sets/gets the spacing, in pixels, between lines of text. This
setting will be used in all subsequent calls to the text() function.

**รูปแบบการใช้งาน**

textLeading ( leading )

**พารามิเตอร์**

- ``leading``  Number: the size in pixels for spacing between lines
