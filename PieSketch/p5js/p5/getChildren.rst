getChildren()
=============

Returns all of the element's children as an array of p5.XML objects. When
the name parameter is specified, then it will return all children that match
that name.

**รูปแบบการใช้งาน**

getChildren ( [name] )

**พารามิเตอร์**

- ``name``  : element name
