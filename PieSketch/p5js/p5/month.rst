month()
=======

p5.js communicates with the clock on your computer. The month() function
returns the current month as a value from 1 - 12.

**รูปแบบการใช้งาน**

month ( )

.. raw:: html

	<script type="text/p5" data-autoplay data-hide-sourcecode>
	var m = month();
	text("Current month: \n" + m, 5, 50);
	</script>
