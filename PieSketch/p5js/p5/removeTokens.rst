removeTokens()
==============

<p>Removes any of the specified characters (or "tokens").</p>
 <p>If no column is specified, then the values in all columns and
 rows are processed. A specific column may be referenced by
 either its ID or title.</p>

**รูปแบบการใช้งาน**

removeTokens ( chars, [column] )

**พารามิเตอร์**

- ``chars``  : String listing characters to be removed

- ``column``  : Column ID (number) or name (string)
