clear()
=======

clears color and depth buffers
with r,g,b,a

**รูปแบบการใช้งาน**

clear ( r, g, b, a )

**พารามิเตอร์**

- ``r``  : normalized red val.

- ``g``  : normalized green val.

- ``b``  : normalized blue val.

- ``a``  : normalized alpha val.
