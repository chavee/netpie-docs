minute()
========

p5.js communicates with the clock on your computer. The minute() function
returns the current minute as a value from 0 - 59.

**รูปแบบการใช้งาน**

minute ( )

.. raw:: html

	<script type="text/p5" data-autoplay data-hide-sourcecode>
	var m = minute();
	text("Current minute: \n" + m, 5, 50);
	</script>
