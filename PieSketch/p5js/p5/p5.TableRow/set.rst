set()
=====

Stores a value in the TableRow's specified column.
 The column may be specified by either its ID or title.

**รูปแบบการใช้งาน**

set ( column, value )

**พารามิเตอร์**

- ``column``  String,Number: Column ID (Number) or Title (String)

- ``value``  String,Number: The value to be stored
