getString()
===========

Retrieves an String value from the TableRow's specified
 column. The column may be specified by either its ID or
 title.

**รูปแบบการใช้งาน**

getString ( column )

**พารามิเตอร์**

- ``column``  String,Number: columnName (string) or ID (number)


**ค่าที่ส่งออกมา**

- String: String
