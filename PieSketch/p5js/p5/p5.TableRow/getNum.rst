getNum()
========

Retrieves a Float value from the TableRow's specified
 column. The column may be specified by either its ID or
 title.

**รูปแบบการใช้งาน**

getNum ( column )

**พารามิเตอร์**

- ``column``  String,Number: columnName (string) or ID (number)


**ค่าที่ส่งออกมา**

- Number: Float Floating point number
