noStroke()
==========

Disables drawing the stroke (outline). If both noStroke() and noFill()
are called, nothing will be drawn to the screen.

**รูปแบบการใช้งาน**

noStroke ( )

.. raw:: html

	<script type="text/p5" data-autoplay data-hide-sourcecode>
	noStroke();
	rect(20, 20, 60, 60);


	</script>
