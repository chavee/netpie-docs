volume()
========

Sets volume for this HTML5 media element. If no argument is given,
returns the current volume.

**รูปแบบการใช้งาน**

volume ( [val] )

**พารามิเตอร์**

- ``val``  Number: volume between 0.0 and 1.0


**ค่าที่ส่งออกมา**

- Number,p5.MediaElement: current volume or p5.MediaElement
