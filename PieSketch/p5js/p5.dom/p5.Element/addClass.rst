addClass()
==========

Adds specified class to the element.

**รูปแบบการใช้งาน**

addClass ( class )

**พารามิเตอร์**

- ``class``  String: name of class to add


**ค่าที่ส่งออกมา**

- Object,p5.Element: 


.. raw:: html

	<script type="text/p5" data-autoplay data-hide-sourcecode>
	var div = createDiv('div');
	div.addClass('myClass');

	</script>
