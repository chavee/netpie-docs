	VIDEO : CAMERA STUFF <VIDEO>
	addElement() : ผู้ช่วยสำหรับสร้างวิธีการ <addElement>
	createA() : สร้าง   องค์ประกอบใน DOM สำหรับใส่ไฮเปอร์ลิงก์ ผนวกเข้ากับโหนดคอนเทนเนอร์หากมีการระบุไว้มิฉะนั้นจะต่อท้ายกับเนื้อหา <createA>
	createAudio() : สร้างองค์ประกอบ HTML5  ที่ซ่อนไว้ใน DOM เพื่อการเล่นเสียงที่เรียบง่าย ผนวกเข้ากับโหนดคอนเทนเนอร์หากมีการระบุไว้มิฉะนั้นจะต่อท้ายกับเนื้อหา พารามิเตอร์ตัวแรกอาจเป็นเส้นทางสตริงเดี่ยวไปยังไฟล์เสียงหรืออาร์เรย์ของเส้นทางสตริงไปยังรูปแบบต่างๆของเสียงเดียวกัน วิธีนี้มีประโยชน์ในการทำให้มั่นใจว่าเสียงของคุณสามารถเล่นผ่านเบราว์เซอร์ต่างๆได้เนื่องจากแต่ละรูปแบบจะสนับสนุนรูปแบบต่างๆ ดู หน้านี้สำหรับข้อมูลเพิ่มเติมเกี่ยวกับรูปแบบที่สนับสนุน <createAudio>
	createButton() : สร้างอิลิเมนต์   ใน DOM ใช้ขนาด. <createButton>
	createCapture() :  สร้าง  องค์ประกอบใหม่ที่มีฟีดเสียง / วิดีโอจากเว็บแคม ซึ่งสามารถวาดลงบนผืนผ้าใบโดยใช้วิดีโอ ()  คุณสมบัติเฉพาะของฟีดอาจส่งผ่านไปได้ในออบเจกต์ Constraints ดู ข้อมูลจำเพาะของ W3C สำหรับคุณสมบัติที่เป็นไปได้ โปรดทราบว่าเบราเซอร์ทุกตัวไม่สามารถใช้งานได้ทั้งหมด  หมายเหตุด้านความปลอดภัย: ข้อกำหนดด้านความปลอดภัยของเบราเซอร์ใหม่จำเป็นต้องใช้ getUserMedia ซึ่งอยู่เบื้องหลัง createCapture () เฉพาะเมื่อคุณใช้งานโค้ดภายในเครื่องเท่านั้นหรือบน HTTPS เรียนรู้เพิ่มเติม ที่นี่ และ ที่นี่  <createCapture>
	createCheckbox() : สร้างอิลิเมนต์   ใน DOM การเรียกใช้. <createCheckbox>
	createDiv() : สร้างอิลิเมนต์   ใน DOM พร้อมด้วย HTML ภายใน ผนวกเข้ากับโหนดคอนเทนเนอร์หากมีการระบุไว้มิฉะนั้นจะต่อท้ายกับเนื้อหา <createDiv>
	createElement() : สร้างอิลิเมนต์ด้วยแท็กที่ระบุใน DOM พร้อมด้วยเนื้อหาที่กำหนด ผนวกเข้ากับโหนดคอนเทนเนอร์หากมีการระบุไว้มิฉะนั้นจะต่อท้ายกับเนื้อหา <createElement>
	createFileInput() : สร้างอิลิเมนต์   ใน DOM ของชนิด 'file' ซึ่งจะช่วยให้ผู้ใช้สามารถเลือกไฟล์ในเครื่องสำหรับใช้ในการร่าง <createFileInput>
	createImg() : สร้างอิลิเมนต์  ใน DOM พร้อมด้วย src และข้อความสำรอง ผนวกเข้ากับโหนดคอนเทนเนอร์หากมีการระบุไว้มิฉะนั้นจะต่อท้ายกับเนื้อหา <createImg>
	createInput() : สร้างอิลิเมนต์   ใน DOM สำหรับป้อนข้อความ ใช้ขนาด. <createInput>
	createMedia() : VIDEO STUFF <createMedia>
	createP() : สร้างอิลิเมนต์   ใน DOM พร้อมด้วย HTML ภายใน ใช้สำหรับข้อความความยาววรรค ผนวกเข้ากับโหนดคอนเทนเนอร์หากมีการระบุไว้มิฉะนั้นจะต่อท้ายกับเนื้อหา <createP>
	createRadio() : สร้างองค์ประกอบปุ่ม   ของฟีดเดอร์ใน DOM สามารถใช้ตัวเลือก. <createRadio>
	createSelect() : สร้างเมนูแบบเลื่อนลง   ใน DOM นอกจากนี้ยังช่วยในการกำหนดวิธีการเลือกกล่องเพื่อ p5.Element เมื่อเลือกกล่องเลือกที่มีอยู่ <createSelect>
	createSlider() : สร้างตัวเลื่อน   องค์ประกอบใน DOM ใช้. <createSlider>
	createSpan() : สร้างองค์ประกอบ   ใน DOM พร้อมด้วย HTML ภายในที่กำหนด ผนวกเข้ากับโหนดคอนเทนเนอร์หากมีการระบุไว้มิฉะนั้นจะต่อท้ายกับเนื้อหา <createSpan>
	createVideo() : สร้างองค์ประกอบ  HTML5 ใน DOM เพื่อการเล่นเสียง / วิดีโอที่เรียบง่าย แสดงโดยค่าเริ่มต้นสามารถซ่อนด้วย. <createVideo>
	getContainer() : ฟังก์ชันตัวช่วยสำหรับการเลือกและเลือกทั้งหมด <getContainer>
	removeElements() : ลบองค์ประกอบทั้งหมดที่สร้างโดย p5 ยกเว้นองค์ประกอบ canvas / graphics ที่สร้างโดย createCanvas หรือ createGraphics ตัวจัดการเหตุการณ์จะถูกลบออกและจะลบองค์ประกอบออกจาก DOM <removeElements>
	select() : ค้นหาหน้าเว็บสำหรับองค์ประกอบที่มี ID ชั้นเรียนหรือชื่อแท็ก (ใช้คำนำหน้า '#' หรือ '.' เพื่อระบุ ID หรือคลาสตามลำดับและไม่มีสำหรับแท็ก) และส่งคืนค่าเป็น p5.Element ถ้าชื่อชั้นหรือชื่อแท็กมีมากกว่า 1 องค์ประกอบจะส่งคืนเฉพาะองค์ประกอบแรกเท่านั้น โหนด DOM สามารถเข้าถึงได้โดยใช้ .elt ส่งกลับค่า null ถ้าไม่มีพบ นอกจากนี้คุณยังสามารถระบุคอนเทนเนอร์เพื่อค้นหาภายใน <select>
	selectAll() : ค้นหาหน้าเว็บสำหรับองค์ประกอบที่มีชั้นหรือชื่อแท็ก (ใช้คำนำหน้า '.' เพื่อระบุคลาสและไม่มีคำนำหน้าสำหรับแท็ก) และส่งกลับค่าเป็น p5.Elements ในอาร์เรย์ โหนด DOM สามารถเข้าถึงได้โดยใช้ .elt ส่งกลับอาร์เรย์ที่ว่างเปล่าหากไม่มีพบ นอกจากนี้คุณยังสามารถระบุคอนเทนเนอร์เพื่อค้นหาภายใน <selectAll>
	wrapElement() : ฟังก์ชันช่วยเหลือสำหรับ getElement และ getElements <wrapElement>

p5.Element
^^^^^^^^^^

.. toctree::
	addClass() : เพิ่มคลาสที่ระบุไปยังองค์ประกอบ <p5.Element/addClass>
	attribute() : เพิ่มแอ็ตทริบิวต์ใหม่หรือเปลี่ยนแปลงค่าของแอ็ตทริบิวต์ที่มีอยู่ในอิลิเมนต์ที่ระบุ ถ้าไม่มีค่าระบุส่งกลับค่าของแอ็ตทริบิวต์ที่กำหนดหรือเป็นค่าว่างหากไม่มีการตั้งค่าแอตทริบิวต์ <p5.Element/attribute>
	center() : กำหนดองค์ประกอบ p5 ทั้งแนวตั้งหรือแนวนอนหรือทั้งสองแบบเทียบกับส่วนที่เป็นแม่หรือตามร่างกายหากองค์ประกอบไม่มีผู้ปกครอง ถ้าอาร์กิวเมนต์ไม่ผ่าน Element ถูกจัดแนวทั้งแนวตั้งและแนวนอน <p5.Element/center>
	child() : แนบองค์ประกอบดังกล่าวกับเด็กไว้กับผู้ปกครองที่ระบุ ยอมรับ ID สตริงโหนด DOM หรือ p5.Element หากไม่มีการระบุอาร์เรย์จะมีการส่งคืนอาร์เรย์ของโหนด DOM ของเด็ก <p5.Element/child>
	hide() : ซ่อนองค์ประกอบปัจจุบัน การตั้งค่าการแสดงผล: ไม่มีสำหรับสไตล์ <p5.Element/hide>
	html() : หากอาร์กิวเมนต์ได้รับให้กำหนด HTML ภายในขององค์ประกอบแทนที่ html ที่มีอยู่ ถ้าความจริงถูกรวมเป็นอาร์กิวเมนต์ที่สอง html จะถูกผนวกแทนการแทนที่ html ที่มีอยู่ ถ้าไม่มีอาร์กิวเมนต์ให้ส่งกลับ HTML ภายในขององค์ประกอบ <p5.Element/html>
	position() : ตั้งค่าตำแหน่งขององค์ประกอบที่สัมพันธ์กับ (0, 0) ของหน้าต่าง กำหนดตำแหน่ง: คุณสมบัติสัมบูรณ์และด้านซ้ายและด้านบนของสไตล์ ถ้าไม่มีอาร์กิวเมนต์ให้ส่งกลับตำแหน่ง x และ y ของอิลิเมนต์ในออบเจ็กต์ <p5.Element/position>
	remove() : นำองค์ประกอบออกและรีจีสเตอร์ผู้ฟังทั้งหมด <p5.Element/remove>
	removeAttribute() : ลบแอตทริบิวต์ในองค์ประกอบที่ระบุ <p5.Element/removeAttribute>
	removeClass() : ลบคลาสที่ระบุออกจากองค์ประกอบ <p5.Element/removeClass>
	show() : แสดงองค์ประกอบปัจจุบัน การตั้งค่าการแสดงผล: ป้องกันสไตล์ <p5.Element/show>
	size() : ตั้งค่าความกว้างและความสูงขององค์ประกอบ AUTO สามารถใช้เพื่อปรับขนาดได้เพียงอย่างเดียวเท่านั้น ถ้าอาร์กิวเมนต์ไม่ได้รับความกว้างและความสูงขององค์ประกอบในวัตถุ <p5.Element/size>
	style() : ตั้งค่าคุณสมบัติสไตล์ (css) ที่ระบุ (อาร์กิวเมนต์ที่ 1) ขององค์ประกอบด้วยค่าที่กำหนด (อาร์เรย์ที่ 2) ถ้าอาร์กิวเมนต์เดียวถูกกำหนดให้. <p5.Element/style>
	value() : ทั้งส่งกลับค่าขององค์ประกอบถ้าอาร์กิวเมนต์ไม่ได้รับหรือกำหนดค่าขององค์ประกอบ <p5.Element/value>

p5.MediaElement
^^^^^^^^^^^^^^^

.. toctree::
	addCue() : กำหนดเวลากิจกรรมเพื่อเรียกใช้ทุกครั้งที่ MediaElement (เสียง / วิดีโอ) ถึงจุดคิวการเล่น ยอมรับฟังก์ชันการโทรกลับเวลา (เป็นวินาที) ที่จะเรียกใช้การโทรกลับและพารามิเตอร์ที่เป็นตัวเลือกสำหรับการโทรกลับ เวลาจะถูกส่งผ่านไปเป็นพารามิเตอร์แรกในฟังก์ชัน callback และ param จะเป็นพารามิเตอร์ที่สอง <p5.MediaElement/addCue>
	autoplay() : ตั้งค่าองค์ประกอบสื่อ HTML5 เพื่อเล่นอัตโนมัติหรือไม่ <p5.MediaElement/autoplay>
	clearCues() : ลบการติดต่อกลับทั้งหมดที่ได้รับการกำหนดเวลาไว้ตามวิธี addCue <p5.MediaElement/clearCues>
	connect() : ส่งเอาต์พุตเสียงของอิลิเมนต์นี้ไปยังออบเจ็กต์ audioNode หรือ p5.sound ที่ระบุ หากไม่มีองค์ประกอบใดให้เชื่อมต่อกับเอาต์พุตหลักของ p5 การเชื่อมต่อดังกล่าวถูกสร้างขึ้นเมื่อใช้วิธีนี้เป็นครั้งแรก การเชื่อมต่อทั้งหมดจะถูกเอาออกโดยใช้. <p5.MediaElement/connect>
	disconnect() : ยกเลิกการเชื่อมต่อเส้นทางเสียงของเว็บทั้งหมดรวมถึงการส่งออกข้อมูลหลัก วิธีนี้มีประโยชน์หากคุณต้องการกำหนดเส้นทางการแสดงผลผ่านทางเอฟเฟ็กต์เสียงอีกครั้ง <p5.MediaElement/disconnect>
	duration() : ส่งกลับระยะเวลาขององค์ประกอบสื่อ HTML5 <p5.MediaElement/duration>
	hideControls() : ซ่อนการควบคุม mediaElement ดีฟอลต์ <p5.MediaElement/hideControls>
	isModified() : helper method สำหรับโหมด web GL เพื่อดูว่าองค์ประกอบได้รับการแก้ไขแล้วหรืออาจจำเป็นต้องอัพโหลดใหม่ไปยังหน่วยความจำเนื้อระหว่างเฟรม <p5.MediaElement/isModified>
	loop() : ตั้ง 'ลูป' เป็น true สำหรับองค์ประกอบสื่อ HTML5 และเริ่มเล่น <p5.MediaElement/loop>
	noLoop() : ตั้ง 'ลูป' เป็นเท็จสำหรับองค์ประกอบสื่อ HTML5 Element จะหยุดเมื่อถึงจุดสิ้นสุด <p5.MediaElement/noLoop>
	onended() : กำหนดเวลากิจกรรมที่จะเรียกเมื่อองค์ประกอบเสียงหรือวิดีโอถึงจุดสิ้นสุด หากองค์ประกอบกำลังวนลูปจะไม่มีการเรียกใช้ องค์ประกอบถูกส่งผ่านไปเป็นอาร์กิวเมนต์ที่เรียกใช้การเรียกกลับ <p5.MediaElement/onended>
	pause() : หยุดชั่วคราวองค์ประกอบสื่อ HTML5 <p5.MediaElement/pause>
	play() : เล่นองค์ประกอบสื่อ HTML5 <p5.MediaElement/play>
	removeCue() : ลบการโทรกลับโดยใช้รหัส ID ถูกส่งคืนโดยเมธอด addCue <p5.MediaElement/removeCue>
	setModified() : วิธีช่วยเหลือสำหรับโหมดเว็บ GL เพื่อระบุว่ามีการเปลี่ยนแปลงองค์ประกอบหรือไม่เปลี่ยนแปลงตั้งแต่อัปโหลดครั้งล่าสุด gl texture upload จะทำให้ค่านี้เป็น false หลังจากอัปโหลดพื้นผิว หรืออาจตั้งค่าเป็นจริงหากเมทาดาทามีให้ใช้งานได้ แต่ไม่มีข้อมูลพื้นผิวจริงที่มีอยู่ <p5.MediaElement/setModified>
	showControls() : แสดงตัวควบคุม MediaElement เริ่มต้นตามที่กำหนดโดยเว็บเบราเซอร์ <p5.MediaElement/showControls>
	speed() : หากไม่ได้รับอาร์กิวเมนต์ให้ส่งคืนความเร็วในการเล่นในปัจจุบันขององค์ประกอบ พารามิเตอร์ความเร็วกำหนดความเร็วที่ 2.0 จะเล่นองค์ประกอบสองครั้งเร็ว 0.5 จะเล่นที่ความเร็วครึ่งหนึ่งและ -1 จะเล่นองค์ประกอบในความเร็วปกติในการย้อนกลับ (โปรดทราบว่าเบราว์เซอร์ไม่สนับสนุนการเล่นย้อนหลังและแม้ว่าจะมี การเล่นอาจไม่ราบรื่น) <p5.MediaElement/speed>
	stop() : หยุดองค์ประกอบสื่อ HTML5 (ตั้งเวลาปัจจุบันเป็นศูนย์) <p5.MediaElement/stop>
	time() : หากไม่ได้รับอาร์กิวเมนต์ให้ส่งคืนเวลาปัจจุบันของอิลิเมนต์ ถ้ามีอาร์กิวเมนต์ระบุเวลาปัจจุบันของอิลิเมนต์ถูกตั้งค่าไว้ <p5.MediaElement/time>
	volume() : ตั้งค่าปริมาณสำหรับองค์ประกอบสื่อ HTML5 ถ้าไม่ได้รับอาร์กิวเมนต์ให้ส่งคืนไดรฟ์ข้อมูลปัจจุบัน <p5.MediaElement/volume>
